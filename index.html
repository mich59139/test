Guide d’utilisation

Exports

  • 📋 Copier : copie les lignes filtrées (TSV).
  • ⬇️ CSV : télécharge un CSV des lignes filtrées.
  • ⬇️ Excel : télécharge un .xlsx (généré côté navigateur).
  • 🖨️ Imprimer : feuille A4 optimisée.

Astuces

  • Mobile : touchez une ligne pour l’éditer. Enter valide, Échap annule.
  • Recherche : saisie délayée (plus fluide).
  • En‑tête collant (desktop) : titres visibles au défilement.

Rechercher : utilisez le champ « Recherche » pour chercher dans tous les champs (titre, auteurs, villes, thèmes, période…).

Filtres : choisissez une Année pour restreindre les résultats. Le menu Numéro se mettra automatiquement à jour pour ne proposer que les numéros existants dans cette année.

Trier : cliquez sur les en-têtes de colonnes (Année, Numéro, Titre, etc.). Cliquez à nouveau pour inverser l’ordre.

Pagination : utilisez « Afficher » pour changer le nombre de lignes visibles (50, 100, Tout) puis naviguez avec Précédent/Suivant.

Ajouter un article : bouton « ➕ Ajouter un article ». Les listes déroulantes proposent automatiquement les valeurs déjà présentes (sans doublons).

Édition/Suppression : double‑cliquez sur une ligne pour éditer. Bouton 🗑️ pour supprimer une ligne.
