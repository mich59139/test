Guide d’utilisation

📜 Objectif

L’application permet de consulter le catalogue d’articles (recherche, filtres, navigation).

🔍 Fonctions disponibles pour tous

  • Recherche : tapez un mot-clé (titre, auteur, ville, thème…).
  • Filtres :
    • Année : limite aux articles de l’année choisie.
    • Numéro : s’adapte automatiquement à l’année sélectionnée.
    • Afficher : nombre d’articles par page.
  • Navigation : utilisez ◀ Précédent / Suivant ▶. Le compteur indique la page et le total.

✏️ Fonctions réservées à l’administrateur

Pour fiabiliser la base, les fonctions d’édition sont provisoirement réservées à l’administrateur :

  • Ajout / Modification / Suppression d’articles
  • Enregistrement dans GitHub (commits)
  • Mode brouillon et enregistrement groupé

🖌️ Conseils d’affichage

  • Si le texte est petit : Ctrl + + (Windows) ou ⌘ + + (Mac).
  • Sur smartphone, utilisez le mode paysage.

🆘 Assistance

En cas de question ou pour demander une mise à jour, contactez l’administrateur.
